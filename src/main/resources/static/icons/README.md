# PWA 아이콘 안내

이 폴더에는 PWA(Progressive Web App)에 필요한 아이콘 파일들이 위치합니다.

## 필요한 아이콘 크기

다음 크기의 아이콘 파일들을 이 폴더에 추가해주세요:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## 아이콘 생성 방법

1. 온라인 PWA 아이콘 생성기 사용:
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/

2. 원본 이미지(최소 512x512px)를 업로드하면 자동으로 모든 크기의 아이콘이 생성됩니다.

3. 생성된 아이콘 파일들을 이 폴더에 복사하세요.

## 주의사항

- 아이콘은 투명 배경을 가진 PNG 파일이어야 합니다.
- 모든 크기의 아이콘이 준비되어야 PWA가 정상적으로 작동합니다.
